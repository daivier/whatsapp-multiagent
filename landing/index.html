Perguntas frequentes

Preciso instalar alguma coisa? +

Não. É um painel web — funciona no navegador do PC ou celular. Você só escaneia um QR Code uma vez para conectar o WhatsApp.

Uso o número que já tenho? +

Sim. Conecta o número de WhatsApp que a sua empresa já usa. Não precisa de chip novo nem número novo.

Meu número pode ser banido? +

O sistema usa uma conexão não oficial (como o WhatsApp Web). Banimentos são raros quando se respeita o uso normal (sem spam em massa), mas é uma possibilidade que orientamos a mitigar. Falamos sobre isso na configuração.

Os dados ficam seguros? +

Cada cliente tem o seu ambiente isolado, com a sua própria base de dados. As conversas ficam guardadas no servidor — se um atendente sair, o histórico permanece com você.

Como funciona o teste grátis? +

1 mês completo, sem cartão de crédito e sem compromisso. A gente configura tudo para você. Se não servir, é só desligar.

Pronto para organizar o seu atendimento?

Teste 1 mês grátis. A gente configura tudo e você vê funcionando com o seu próprio número.
